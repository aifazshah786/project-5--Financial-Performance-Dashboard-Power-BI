# project 5- Financial Overview Dashboard – Power BI
Developed a fully interactive Financial Performance Dashboard using Power BI to help stakeholders monitor financial health and track key KPIs. The dashboard analyzes Revenue, Profitability, Cash Flow, Sales Performance, and Budget vs Actual using dynamic visualizations, DAX measures, and interactive filters to support data-driven decisions.
